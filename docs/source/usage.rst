Usage
=====

.. _installation:

Installation
------------

- Start Blender
- Go to "Edit" -> “Preferences” –> “Add-ons” –> “Install” –> open the add-on zip file
- Activate the add-on

.. _how_to_use:

How to use
----------

- When using PTK I recommend using **points** as particles and then instantiating mesh on those points using the repective nodes.
